# financial-dashboard-frontend
Frontend-dashboard для донатного сервиса с авторизацией, управлением формами, загрузкой изображений, фильтрацией данных и визуализацией аналитики. Проект реализован как масштабируемое SPA на React и TypeScript с упором на архитектуру, качество кода и работу с production-кейсам.
